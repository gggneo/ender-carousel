Build your HTML structure for the carousel

For the vertical carousel you can use the same HTML as you've seen in the basic tutorial. For the most simple CSS possible I've replaced the previous pager.

Just for the clarity, here is a recap of the elements:

  • The root element, you will call $.carousel on it. Every elements need to be inside this element. It's .carousel in the example code
  • Window element - it will be the mask of the animated elem wrapper. Defaults to: .window
  • Item wrapper - defaults to the first child of the window element.
  • Items: a selector to determine the elements. Defaults to: li
  • Next pager - defaults to: .next
  • Previous pager - defaults to: .prev

Example HTML

<div class="carousel">
    <a class="prev">Previous</a>
    <div class="window">
        <ul class="clr">
            <li>
                <img src="#Your Image URL#" alt="#Img desc#" width="100" height="100">
            </li>
            [...]
        </ul>
    </div> <!-- .window -->
    <a class="next">Next</a>
</div> <!-- .carousel -->

Add the necessary CSS rules to your stylesheet

The CSS for this type of carousel is simpler than the horizontal one. Here is the core of the CSS, but you should check the ender-carousel-vertical.css file for the full version of the CSS.

Example CSS

.carousel > .window {
    overflow: hidden;
    position: relative;
    height: 370px;
}

/*
    This is the item wrapper,
    it will be animated.
*/
.carousel ul {
    list-style: none;
    height: 10000px;
    top: 0;
    left: 0;
    position: absolute;
}

/*
    These are the items
*/
.carousel li {
    margin: 3px 0 0;
}

.carousel li:first-child {
    margin: 0;
}

Initialize the ender-carousel plugin

Now the easy part. You can add a settings object to the carousel method, check the configuration for the details. Please note that currently the keyboard support is buggy if the carousel is vertical.

$(document).ready(function () {
    $(".carousel").carousel({
        vertical: true
    });
});
